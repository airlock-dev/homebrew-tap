# Airlock Homebrew Tap

Homebrew formulae and casks for [Airlock](https://airlock.bot).

## Install

```bash
brew install airlock-dev/tap/airlock-companion
```

## Available casks

| Cask | Description |
|------|-------------|
| `airlock-companion` | macOS menu bar app for HITL approvals |
